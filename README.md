# MEVN-MyFirstProject

## How to run?

### Server
```shell
cd server

// install all packages
npm install || yarn

// run development server
npm run dev || yarn dev

// run production server
npm run serve || yarn serve
```

### Client
```shell
cd client

// install all packages
npm install || yarn

// run development server
npm run dev || yarn dev

// build app to production
npm run build || yarn build

// start production server
npm run serve || yarn serve
```
